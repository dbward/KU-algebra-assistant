# KU-algebra-assistant
A chatbot connected to an open source textbook for tutoring students on College Algebra.

## Math 101 Online – University of Kansas, Fall 2025

### Overview
An AI-powered tutoring assistant for Math 101 College Algebra, 
built using the Anthropic Claude API. Designed to support students 
through Socratic guidance rather than direct answers.

### Features
- Course schedule and due date lookup
- Socratic tutoring for algebra concepts
- Study guide and practice problem generation
- Course policy reference

### Technical Setup
- **Frontend:** HTML/CSS/JavaScript hosted on GitHub Pages
- **API Proxy:** Cloudflare Worker (handles CORS and authentication)
- **AI Model:** Anthropic Claude Sonnet
- **Textbook:** OpenStax College Algebra 2e (Chapters 1–7)
